# 项目记忆

## 项目概述
- **项目名称**: 时势量化工作室独立站
- **技术栈**: Next.js + Tailwind CSS
- **部署平台**: Cloudflare Pages (通过 GitHub 自动部署)
- **仓库**: https://github.com/alexandra2472/timingquant

## 网站结构

### 页面板块（按顺序）
1. **Hero** - 首页横幅
2. **Features** - 核心优势
3. **Products** - 产品服务
4. **MonthlyReview** - 月度复盘 ⭐ 新增
5. **Cases** - 成功案例（轮播图展示）
6. **Pricing** - 定价方案
7. **About** - 关于我们
8. **Footer** - 页脚

### 导航菜单
- 首页、特色、产品、月度复盘、案例、定价、关于
- 支持中英文切换
